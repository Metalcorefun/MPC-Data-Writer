Руководство пользователя MPC Data Writer
==========================================================

Для чего эта программа?
----------------------------------------------------------
Данный продукт предназначен для синхронной записи видео, аудио и параметрических данных человека в реальном времени с использованием специфической аппаратной конфигурации.

Системные требования
----------------------------------------------------------

### Требования к программному окружению ###

ОС Windows 8.1 и выше.

### Требования к аппаратному окружению ###

Необходимо наличие:
1. Подключенной к локальной сети IP-камеры.
2. Встроенной или дискретной звуковой карты и микрофона.
3. Полиграфа Поликониус Диана-7М.

Порядок установки
----------------------------------------------------------
1. Распаковать содержимое репозитория в любую папку.
2. Распаковать содержимое папки "*Папка с распакованным репозиторием*\DataWriter\DataWriter\bin\Release" в любую папку.
3. Использовать и наслаждаться бесперебойной работой программы:)
